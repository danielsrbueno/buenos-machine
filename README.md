# 🔢 Bueno's Machine

Conversor de bases numéricas simples e direto ao ponto. Digite um número em qualquer base e todas as outras são atualizadas automaticamente.

**Bases suportadas:** Decimal · Octal · Hexadecimal · Binário

---

## ✨ Funcionalidades

- **Conversão em tempo real** — qualquer campo atualiza os demais enquanto você digita
- **Validação de entrada** — feedback visual imediato para caracteres inválidos
- **Copiar com um clique** — botão de cópia em cada campo com notificação animada
- **Área de texto para binário** — o campo binário expande automaticamente conforme o número cresce
- **Interface responsiva** — funciona bem em desktop e mobile

---

## 🚀 Como usar

Não há instalação necessária. Basta abrir o arquivo `index.html` direto no navegador.

```bash
# Clone o repositório
git clone https://github.com/danielsrbueno/buenos-machine.git

# Abra o arquivo no navegador
open index.html
```

Ou acesse diretamente: [machine.danbueno.com](https://machine.danbueno.com/)

| Tecnologia | Uso |

## 📐 Bases suportadas

| Base | Nome | Caracteres válidos | Exemplo |
|---|---|---|---|
| 2 | Binário | `0`, `1` | `101010` |
| 8 | Octal | `0`–`7` | `52` |
| 10 | Decimal | `0`–`9` | `42` |
| 16 | Hexadecimal | `0`–`9`, `a`–`f` | `2A` |
